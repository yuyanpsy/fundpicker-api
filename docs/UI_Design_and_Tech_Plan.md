# 智选基金 (FundPicker) — UI设计 & 技术方案

## 一、产品概述

一款基于 AI 预测的基金筛选 Android App，帮助用户从数千只基金中快速找到上涨概率最大的产品。

核心价值：**数据驱动 + AI预测 + 简洁体验**

---

## 二、信息架构

```
App
├── 首页 (AI推荐)
│   ├── AI精选榜单（上涨概率排序）
│   ├── 市场情绪指标
│   └── 热门板块轮动
├── 发现 (基金市场)
│   ├── 全部基金列表（搜索/筛选）
│   ├── 分类浏览（股票型/混合型/债券型/指数型/QDII）
│   └── 排行榜（日/周/月/年涨幅）
├── 基金详情页
│   ├── 基本信息
│   ├── 净值走势图（可切换时间维度：7天/30天/3月/6月/1年/3年/全部）
│   ├── AI预测面板（可切换预测周期：7天/30天/3月/6月/1年）
│   │   └── AI模型对比（多算法/多参数结果并排对比）
│   ├── 持仓分析（可切换持仓报告期：最新/上季/半年前/1年前）
│   └── 基金经理信息
├── 模拟持仓 ★新增
│   ├── 模拟组合总览（总资产/总收益/日收益）
│   ├── 持仓列表（买入价/现价/盈亏/仓位占比）
│   ├── 模拟买入/卖出操作
│   ├── 收益走势图（可切换时间维度）
│   └── 交易记录
├── AI实验室 ★新增
│   ├── 算法对比面板（LSTM vs LightGBM vs 集成模型）
│   ├── 参数调节器（滑块调整模型权重/窗口期/特征组合）
│   ├── 对比结果可视化（多曲线叠加图）
│   └── 历史回测对比
├── 自选 (我的关注)
│   ├── 自选基金列表
│   ├── 组合收益统计
│   └── 预警通知
└── 我的
    ├── 设置
    ├── 时间维度偏好设置 ★新增
    ├── 风险偏好设置
    └── 关于
```

## 四、交互逻辑

### 4.1 核心交互流程

#### 流程1：首次启动

```
启动App → 闪屏(品牌Logo) → 风险偏好问卷(3题)
  → 保守型: 优先推荐债券型/货币型基金
  → 稳健型: 推荐混合型为主
  → 进取型: 推荐股票型/指数型为主
→ 进入首页(AI推荐已根据偏好排序)
```

#### 流程2：AI推荐浏览

```
首页 → 查看AI精选Top10
  → 点击基金卡片 → 进入详情页
    → 查看净值走势
    → 查看AI预测面板(概率/置信度/因子)
    → 点击"关注" → 加入自选
  → 返回首页继续浏览
```

#### 流程3：主动搜索筛选

```
底部Tab"发现" → 搜索栏输入基金名称/代码
  → 实时联想匹配
  → 点击结果 → 进入详情页

或: 点击分类标签 → 筛选条件
  → 设置AI预测概率阈值(如≥70%)
  → 设置基金类型/风险等级/规模
  → 确认 → 列表按AI评分排序展示
```

#### 流程4：自选管理

```
自选页 → 查看已关注基金实时数据
  → 左滑单个基金 → 删除
  → 点击"编辑" → 批量管理/排序
  → 查看AI预警通知
    → 点击预警 → 跳转对应基金详情
```

#### 流程5：AI预警推送

```
后台定时任务(每日收盘后) → 计算AI预测
  → 自选基金评分大幅变动(>10%) → 推送通知
  → 用户点击通知 → 打开App → 跳转基金详情
```

#### 流程6：模拟持仓 ★新增

```
入口1: 基金详情页 → 点击"模拟买入" → 弹出买入弹窗
  → 输入金额(或点击快捷金额) → 确认买入
  → Toast提示"买入成功" → 自动跳转模拟持仓页

入口2: 底部Tab"持仓" → 模拟持仓页
  → 查看总资产/总收益/今日收益
  → 切换收益走势时间维度(7天/30天/3月/6月/1年)
  → 查看持仓明细
    → 点击基金 → 跳转基金详情
    → 点击"加仓" → 弹出买入弹窗
    → 点击"卖出" → 弹出卖出弹窗
      → 选择卖出方式(按金额/按份额/全部)
      → 确认卖出 → 更新持仓
  → 点击"模拟买入+" → 搜索基金 → 买入新基金
  → 查看交易记录 → 完整交易流水
```

#### 流程7：AI实验室（模型对比）★新增

```
基金详情页 → AI预测面板 → 点击"AI模型对比"
  → 进入AI实验室页
  → 勾选要对比的算法(LSTM/LightGBM/集成/纯技术/纯情绪)
  → 选择预测周期(7天/30天/3月/6月/1年)
  → 调节参数滑块:
    · LSTM时间窗口(30-120日)
    · 集成权重(各模型权重0-1，自动归一化)
    · 特征组合(勾选/取消)
  → 点击"运行对比分析"
  → 等待计算(显示进度动画)
  → 展示结果:
    · 多曲线叠加图(各模型预测走势)
    · 对比表格(上涨概率/置信度)
    · 历史回测对比(命中率/年化收益)
  → 切换回测区间(3月/6月/1年/3年)
  → 可保存当前参数配置为"我的策略"
```

#### 流程8：时间维度切换 ★新增

```
全局时间维度选项: [7天] [30天] [3月] [6月] [1年] [3年] [全部]

适用场景:
  · 净值走势图: 切换Tab → 图表重新加载对应区间数据
  · AI预测周期: 切换Tab → 重新请求对应周期的预测结果
  · 业绩表现: 切换Tab → 高亮对应行/展示对应区间
  · 持仓报告期: 切换Tab → 加载对应季度的持仓数据
  · 模拟持仓收益: 切换Tab → 收益走势图更新
  · AI回测区间: 切换Tab → 回测数据更新

交互细节:
  · 选中态: 主色填充 + 白色文字
  · 未选中: 灰色描边 + 灰色文字
  · 切换时: 图表区域显示加载骨架屏(0.3s)
  · 用户可在"我的→时间维度偏好"设置默认周期
```

### 6.4 AI 预测模型设计

#### 模型架构：多模型集成

```
输入特征
  │
  ├─→ [LSTM模型] ──→ 时序预测概率 P1
  │   (历史净值序列, 60日窗口)
  │
  ├─→ [LightGBM模型] ──→ 多因子预测概率 P2
  │   (技术指标 + 基本面 + 资金流)
  │
  ├─→ [情绪分析模型] ──→ 情绪得分 P3
  │   (财经新闻 + 政策文本)
  │
  └─→ [集成层: 加权平均] ──→ 最终上涨概率
      W1*P1 + W2*P2 + W3*P3
      (权重通过验证集动态调整)
```

#### 特征工程

```python
# 技术指标特征
features_technical = [
    'MA5', 'MA10', 'MA20', 'MA60',    # 均线
    'MACD', 'MACD_signal', 'MACD_hist', # MACD
    'RSI_14',                           # RSI
    'BOLL_upper', 'BOLL_lower',        # 布林带
    'volume_ratio',                     # 量比
    'turnover_rate',                    # 换手率
]

# 基本面特征
features_fundamental = [
    'fund_size',           # 基金规模
    'manager_years',       # 基金经理从业年限
    'sharpe_ratio',        # 夏普比率
    'max_drawdown',        # 最大回撤
    'sector_momentum',     # 所属板块动量
]

# 资金流特征
features_flow = [
    'net_inflow_1d',       # 1日净流入
    'net_inflow_5d',       # 5日净流入
    'institution_ratio',   # 机构持仓占比
]
```

#### 训练与更新策略

```
训练数据: 近3年历史数据
验证方式: 时间序列交叉验证(Walk-Forward)
更新频率: 每周末全量重训练
每日更新: 增量推理(新数据输入已训练模型)
评估指标: AUC-ROC, 精确率, 回测年化收益
```

#### 多模型对比引擎 ★新增

```
AI实验室的核心是一个可配置的预测引擎，支持:

1. 独立模型运行
   - 每个模型(LSTM/LightGBM/情绪)可独立输出预测结果
   - 用户可选择任意组合进行对比

2. 参数化集成
   - 集成权重可由用户自定义(W1+W2+W3=1，前端自动归一化)
   - LSTM窗口期可调(30/60/90/120日)
   - 特征组合可选(技术/基本面/资金流/宏观)

3. 多周期预测
   - 短期: 7天、30天 → 主要依赖技术面+资金流
   - 中期: 3月、6月 → 技术面+基本面+情绪
   - 长期: 1年 → 基本面+宏观+行业趋势
   - 不同周期自动调整特征权重

4. 回测引擎
   - Walk-Forward回测: 滚动窗口模拟真实预测场景
   - 输出指标: 命中率、年化收益、最大回撤、夏普比率
   - 支持多区间对比(3月/6月/1年/3年)

5. 预测结果缓存
   - 相同参数组合的结果缓存到Redis(TTL=1天)
   - 避免重复计算，提升响应速度
```

## 八、风险与合规

1. 数据源风险：AKShare依赖东方财富等网站接口，可能被限流或变更，需做好降级方案
2. AI预测免责：所有预测页面必须展示风险提示，明确"不构成投资建议"
3. 合规要求：如涉及基金销售/推荐，可能需要基金销售牌照，建议定位为"信息工具"而非"投资顾问"
4. 隐私保护：用户自选数据本地存储为主，服务端仅存储必要数据

### 2026-07-04

#### 架构迁移：Render Cron Jobs → GitHub Actions
- **原因**：Render Cron Job 不支持 Free 计划（起步 $0.00016/分钟），产生约 $0.81/月费用
- **方案**：将 3 个 Cron Job 迁移到 GitHub Actions（$0/月，2000分钟免费额度）
- **新增文件**：
  - `.github/workflows/batch-predict.yml` — 每天 UTC 1:00（北京9:00），仅周一至周五
  - `.github/workflows/daily-snapshot.yml` — 每天 UTC 9:30（北京17:30），仅周一至周五
  - `.github/workflows/daily-verify.yml` — 每天 UTC 10:00（北京18:00），仅周一至周五
  - `requirements-cron.txt` — 轻量依赖文件（pandas, numpy, scikit-learn, joblib, requests, ta）
  - `.gitignore` — Git 忽略规则
- **代码修改**：
  - `supabase_store.py`、`daily_snapshot.py`、`daily_verify.py`：Supabase 配置改为 `os.environ.get()` 读取环境变量，支持 GitHub Secrets 注入
- **保留**：Render Web Service（Free，提供 REST API）、Supabase（Free Tier 数据存储）
- **总费用**：$0/月（Render Web $0 + GitHub Actions $0 + Supabase $0）

#### 系统运行状态
